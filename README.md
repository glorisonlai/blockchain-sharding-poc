# Blockchain Sharding POC

### Implementation of Parallel sharding

### Clone

`git clone https://github.com/glorisonlai/blockchain-sharding-poc`

### Pull from source:

`git pull`

### Setting up venv:
```pip install venv```
```python -m venv ./```

### Run locally:

1. `source Scripts/activate`
2. `npm install`
3. `npm run dev`
